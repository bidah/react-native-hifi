# Privacy Policy

**Plugin:** React Native HiFi
**Author:** Rodrigo Figueroa
**Last updated:** 2026-04-08

## Overview

React Native HiFi is a local development workflow plugin for Claude Code. It provides composable "skills" — structured prompts and instructions — that guide your coding agent through React Native and Expo development tasks.

## Data Collection

React Native HiFi does **not** collect, transmit, store, or process any user data. Specifically:

- **No telemetry:** The plugin does not send any usage data, analytics, or metrics to any server.
- **No network requests:** The plugin makes no outbound network connections. It operates entirely locally on your machine.
- **No personal information:** The plugin does not access, read, or store any personal information, credentials, API keys, or authentication tokens.
- **No tracking:** The plugin does not use cookies, fingerprinting, or any other tracking mechanism.
- **No third-party services:** The plugin does not integrate with or send data to any third-party analytics, advertising, or data-processing services.

## How It Works

The plugin consists entirely of local Markdown files (skills, hooks, commands) that are read by your coding agent during a session. All processing happens locally within your Claude Code environment. The plugin:

1. Provides skill files that Claude Code reads to guide its workflow
2. Defines hooks that run locally in your shell
3. Includes commands that execute in your local development environment

No data leaves your machine as a result of installing or using this plugin.
